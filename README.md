
# DearOne

**Never Miss a Special Moment ❤️**

DearOne is a modern Android application designed to help users remember and celebrate important moments in life. Whether it's a birthday, anniversary, or a special occasion, DearOne keeps everything organized and sends timely reminders so that no important event is ever forgotten.

---

## 📱 About DearOne

DearOne is an offline-first event reminder application built using **Kotlin** and **Jetpack Compose**. The app allows users to manage personal events, receive notifications, track upcoming occasions, and generate personalized wishes.

---

## ✨ Features

### 🎂 Event Management

- Add new events
- Edit existing events
- Delete events
- View detailed event information
- Mark important events as favorites

### 🔔 Smart Notifications

- Automatic reminder notifications
- Event-based alerts
- Timely birthday and anniversary reminders

### 📅 Event Categories

- Birthday
- Anniversary
- Custom Events

### 📊 Dashboard Insights

- Total Events Count
- Birthday Count
- Anniversary Count
- Events This Month

### ❤️ Favorites System

- Mark important people/events as favorites
- Quick access to important reminders

### 🔍 Event Filtering

- View All Events
- Filter Birthdays
- Filter Anniversaries
- Filter Favorites

### ⏳ Countdown Tracking

- Remaining days until event
- Tomorrow indicator
- Today indicator

### 📋 Event Details Screen

- Event information
- Notes
- Remaining days
- Quick edit option
- Quick delete option
- Generate Wish button

### 🎨 Modern UI

- Built with Jetpack Compose
- Responsive layout
- Material Design 3 components
- Clean and user-friendly interface

### 📢 AdMob Integration

- Google AdMob Banner Ads integrated
- Test ads successfully loading
- Monetization ready

### 📱 Offline First

- Works without requiring a cloud database
- Fast and lightweight experience
